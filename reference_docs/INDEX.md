# Reference documents

Included TI reference documents:

- `AWR2944EVM_user_guide_SPRUJ22C.pdf` — AWR2944 EVM hardware guide
- `DCA1000_mmWave_Studio_raw_capture_training.pdf` — DCA1000 capture training deck
- `DCA1000_mmWave_Studio_raw_capture_video_transcript.txt` — Transcript of training video
- `SWRA581B_mmWave_ADC_raw_data_capture_app_report.pdf` — ADC raw data format details
- `mmwaveSensing-FMCW-offlineviewing_0.pdf` — FMCW radar formulas (range, velocity, angle)

Use these for implementation guidance, especially binary formats and setup sequences.
Do not assume all old xWR layouts apply directly to AWR2944 without validation.

The FMCW offline viewing deck is the **source of truth** for radar formulas
used in the derived parameter calculations.
